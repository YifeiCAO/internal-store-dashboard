当前模型架构

Transformer 是 PFC；EC/HPC 是外挂。context 只改地址，不直接给 decoder 答案。

外部输入

  • 当前动作 a_t
  • 上一步感觉 x_(t-1)

无 room、switch、position、place ID、当前 x_t。

Transformer=PFC

  • 32-step window
  • 12-step cyclic signature
  • 16-d retention state

state 只输出 context key。

Neural EC + Place

  • 动作驱动 rotation-grid
  • 256-d sparsemax place
  • 约 6.55 active units

地址 = place × context，共 4096 维。

Covariance-corrected HPC

内容矩阵 [B, 64, 4096]
context covariance [B, 16, 16]
每个 episode 清零，dual-key 写入

调用与输出

  • retrieval 投影回 PFC hidden
  • 同一个 sensory decoder
  • all-step CE

HPC-zero 干预验证调用是否真实。

正式 M1b 只有 episode-local content fast-weight matrix；不存在固定离散 memory slots、按 place index 查表或跨 episode 持久化的内容表。实验性 M1e 曾增加第二套 context fast weight，但因 no-recall 消融无损而被拒绝。
